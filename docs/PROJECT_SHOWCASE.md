## 🎯 核心功能

### 1️⃣ 家庭协作管理 👨‍👩‍👧‍👦
- **家庭创建与管理**：创建家庭、邀请成员加入
- **多视角切换**：个人视角 / 家庭视角一键切换
- **成员资产概览**：家庭主账号可查看所有成员资产分布
- **资产/负债转移**：家庭成员间资产和负债的转移
- **家庭财务档案**：家庭整体财务画像与目标配置

### 2️⃣ 资产管理
- **证券持仓追踪**：支持 A股/港股/美股/日股 多市场
- **实时行情接入**：Tushare 证券数据 API，自动更新行情
- **现金资产管理**：活期存款、定期存款、货币基金、券商现金
- **另类投资**：黄金（饰品/ETF）、实物资产
- **不动产管理**：房产、REITs
- **多币种支持**：CNY/USD/HKD/JPY，实时汇率换算

### 3️⃣ 负债管理
- 信用卡、房贷、车贷、消费贷等分类管理
- 还款计划跟踪
- 负债率(DTI)自动计算

### 4️⃣ 智能资产配置 🤖
- **AI投资建议**：接入 DeepSeek 大模型
- **风险评估**：保守型/稳健型/进取型
- **个性化分析**：基于家庭财务状况定制建议
- **二级资产细化**：具体到地区、产品类型

### 5️⃣ 资产全景与快照 📸
- 每日自动快照（Cron Job）
- 历史趋势折线图（个人/家庭）
- 资产配置饼图
- 地区分布分析
- 收益率对比

### 6️⃣ 数据导入
- Excel批量导入
- 多种数据格式支持
- 智能字段匹配

## 🔐 架构亮点

### 1. 服务层架构（Service Layer）
```
所有业务逻辑集中在 Service 层（9 个服务，7,525 行）
├── PortfolioService       - 投资组合计算（1,937 行）
├── AllocationService      - 资产配置分析（1,589 行）
├── FamilyService          - 家庭管理（1,306 行）
├── DeepSeekService        - AI建议生成（889 行）
├── PriceUpdateService     - 证券行情更新（533 行）
├── SnapshotService        - 资产快照（397 行）
├── ActivityLogService     - 活动日志（375 行）
├── LiabilityService       - 负债管理（286 行）
└── AssetCalculationService - 资产计算（213 行）
```

### 2. 前端零计算原则
- API 返回完整计算数据（市值、盈亏、汇率转换）
- 前端只负责渲染和交互
- 保证数据一致性，避免精度问题

### 3. 多币种统一处理
- 实时汇率 API 集成（支持 CNY/USD/HKD/JPY）
- 24小时智能缓存机制
- CNY 为基准货币统一换算

### 4. AI 提示词工程
- 系统提示词与用户提示词分离
- Token 优化（节省 ~620 tokens/次）
- 结构化 JSON 输出，便于前端渲染

### 5. 家庭协作体系
- 家庭创建、成员邀请（邮箱/链接）
- Owner/Admin/Member 三级角色权限
- 家庭视角下聚合所有成员资产快照
- 成员间资产/负债转移与活动日志
