# 资产计算公式说明

## 📊 首页资产概览指标

### 1. 总资产

```
总资产 = 证券持仓市值 + 证券账户现金 + 现金资产 + 其他资产
```

**组成部分**：
- **证券持仓市值**：Σ(数量 × 现价 × 汇率)
- **证券账户现金**：各证券账户的可用现金余额 × 汇率
- **现金资产**：活期存款 + 定期存款(含利息) + 货币基金(含收益)
- **其他资产**：不动产估值 + 贵金属估值 + 其他资产估值

**汇率说明**：
- 所有外币资产均按实时汇率换算为人民币
- 汇率来源：exchangerate-api.com / frankfurter.app
- 缓存周期：5分钟内使用内存缓存

---

### 2. 今日收益

```
今日收益 = 今日总资产 - 昨日总资产（快照）
今日收益率 = 今日收益 ÷ 昨日总资产 × 100%
```

**说明**：
- 基于历史快照数据计算
- 如果没有昨日快照，会尝试使用最近7天内的快照
- 包含全量家庭资产（证券+现金+其他）
- 如果无历史数据，显示为0

---

### 3. 累计收益

```
累计收益 = 证券收益 + 现金资产收益 + 其他资产收益
收益率 = 累计收益 ÷ 总成本基础 × 100%
```

**各部分计算**：

#### 3.1 证券收益
```
证券收益 = Σ[(现价 - 成本) × 数量 × 汇率]
```
- 成本 = 平均买入价格
- 支持多币种（USD/HKD/CNY）

#### 3.2 现金资产收益
```
货币基金收益 = 本金 × 7日年化收益率 ÷ 365 × 持有天数
定期存款利息 = 本金 × 年利率 ÷ 365 × 已过天数
```
- 货币基金：根据 metadata.yield7Day 计算
- 定期存款：根据 metadata.interestRate 计算
- 活期存款：无收益

#### 3.3 其他资产收益
```
其他资产收益 = 当前估值 - 购买成本
```
- 不动产：使用 metadata.marketValue 或 currentValue
- 贵金属：使用 currentValue

---

### 4. 净资产

```
净资产 = 总资产 - 总负债
```

**负债包含**：
- 房贷
- 车贷
- 消费贷
- 其他贷款

---

### 5. 流动现金

```
流动现金 = 证券账户现金 + 活期存款
```

**说明**：
- 只包含高流动性资金
- 不包含定期存款、货币基金（有赎回限制）

---

## 💹 证券持仓计算

### 单个持仓

```
市值(原币种) = 数量 × 现价
市值(CNY) = 市值(原币种) × 汇率
成本基础 = 数量 × 平均成本 × 汇率
未实现盈亏 = 市值(CNY) - 成本基础
收益率 = 未实现盈亏 ÷ 成本基础 × 100%
```

### 占比计算

```
持仓占比 = 持仓市值(CNY) ÷ 总证券市值 × 100%
```

---

## 💰 现金资产计算

### 活期存款
```
当前价值 = 本金 × 汇率
收益 = 0（无利息）
```

### 定期存款
```
已过天数 = min(当前日期 - 存入日期, 到期日期 - 存入日期)
利息收益 = 本金 × 年利率 ÷ 365 × 已过天数
当前价值 = (本金 + 利息收益) × 汇率
```

### 货币基金
```
持有天数 = 当前日期 - 购买日期
收益 = 本金 × 7日年化收益率 ÷ 365 × 持有天数
当前价值 = (本金 + 收益) × 汇率
```

---

## 🏠 其他资产计算

### 不动产
```
当前价值 = metadata.marketValue 或 currentValue 或 purchasePrice
```
优先级：marketValue > currentMarketPrice > appraisalValue > currentValue > purchasePrice

### 贵金属
```
当前价值 = currentValue × 汇率
```

### 其他
```
当前价值 = (currentValue 或 purchasePrice) × 汇率
```

---

## 🔄 汇率获取

### 数据源优先级
1. exchangerate-api.com
2. frankfurter.app
3. 本地默认汇率（fallback）

### 默认汇率（兜底）
- USD: 7.2
- HKD: 0.92
- CNY: 1

### 缓存策略
- 内存缓存：5分钟
- 数据库缓存：永久（手动刷新时更新）

## 🪙 贵金属市值计算（ALT_GOLD）

### 实时金价计算逻辑

贵金属资产（ALT_GOLD）使用 `goldPriceService` 实时计算当前市值，而不是使用数据库中的购买价格。

#### 金价获取
```typescript
// 基准价格：上海金交所（2026-01-28）
const BASE_PRICES_CNY = {
  gold: 1134,      // 黄金 ~1134 CNY/克
  silver: 8.2,     // 白银 ~8.2 CNY/克
  platinum: 310,   // 铂金 ~310 CNY/克
  palladium: 320,  // 钯金 ~320 CNY/克
};

// 品牌溢价系数
const BRAND_PREMIUM = {
  gold: {
    chow_tai_fook: 1.427,    // 周大福 1618/1134 ≈ 1.427
    lao_feng_xiang: 1.429,   // 老凤祥
    chow_sang_sang: 1.423,   // 周生生
    luk_fook: 1.421,         // 六福
    investment: 1.251,       // 投资金条
  },
};
```

#### 市值计算
```
当前市值 = 重量(克) × 当前金价(元/克)

其中：
- 品牌首饰金价 = 基准金价 × 品牌溢价系数
- 投资金条金价 = 基准金价 × 1.251
- 无品牌金价 = 基准金价（现货价）
```

#### 示例
```
周大福生肖金饰：
- 重量：3.3克
- 买入价：427元/克
- 当前金价：1618元/克（周大福）
- 买入成本：3.3 × 427 = ¥1,409
- 当前市值：3.3 × 1618 = ¥5,340
- 盈亏：¥3,931 (+279%)
```

**数据来源**：
- `goldPriceService.getCurrentPrice()` - 根据 metalType、goldCategory、jewelryBrand 获取实时价格
- 价格每30分钟缓存刷新

## 🏷️ 资产类型分类体系

### Level 1 顶层分类（5大类）

| 代码 | 名称 | 颜色 | 说明 |
|------|------|------|------|
| **EQUITY** | 权益类投资 | #3B82F6 (蓝色) | 股票、ETF、基金等 |
| **CASH** | 现金及现金等价物 | #10B981 (绿色) | 活期存款、定期存款、货币基金、证券账户现金 |
| **FIXED_INCOME** | 固定收益类 | #F59E0B (黄色) | 债券、理财产品等 |
| **REAL_ESTATE** | 不动产类 | #6366F1 (紫色) | 住宅、商业地产等 |
| **ALTERNATIVE** | 另类投资 | #EC4899 (粉色) | 贵金属、收藏品等 |

### 资产分布计算

```
每类占比 = 该类资产总值 ÷ 总资产 × 100%
```

**数据来源**：
- 证券持仓：从 Holding 表读取，按 Security.AssetCategory 分类
- 现金资产：从 Asset 表读取，按 AssetCategory 分类
- 证券账户现金：从 InvestmentAccount.cashBalance 读取，归入 CASH 类
