# 资产分类体系 V2.0 - 优化设计方案

## 📌 设计理念变更

### V1.0 → V2.0 核心变化

#### 变化1：合并权益类和基金类 ⭐
**原因**：
- 基金本质上是权益性投资的工具形式
- 例如：标普500 ETF 本质是投资美国500家公司的权益
- 用户更关心**投资标的所属地**，而非投资工具类型

**变更**：
```
V1.0: 6大分类
├─ 权益类资产 (股票)
└─ 基金类资产 (基金)

V2.0: 5大分类
└─ 权益类投资 (股票+基金合并)
```

#### 变化2：二级分类改为按地区 ⭐
**原因**：
- 资产配置的核心是**地区分散**和**经济体风险分散**
- 拼多多虽在美股上市，但属于中国公司，与中国经济挂钩
- 平安美国50 ETF 虽在中国券商购买，但投资标的是美国公司

**核心逻辑**：
```
交易市场 (region) ≠ 投资标的所属地 (targetCountry)

示例1: 拼多多 (PDD)
  - 交易市场: 美股 (region=US)
  - 公司属地: 中国 (targetCountry=CN)
  - 分类: 权益类投资 > 中国资产 ✅

示例2: 标普500 ETF
  - 交易市场: 港股 (region=HK)
  - 投资标的: 美国 (targetCountry=US)
  - 分类: 权益类投资 > 美国资产 ✅
```

## 📊 V2.0 完整分类体系

### 顶层5大分类

#### 1. 🔵 权益类投资 `#3B82F6`
**定义**：股票、基金等权益性投资，按投资标的所属经济体分类

**二级分类（按地区）**：
- 🇨🇳 **中国资产** (`EQUITY_CN`)
  - A股：茅台、腾讯、比亚迪
  - 中概股：拼多多(PDD)、阿里(BABA)、京东(JD)
  - 投资中国的基金/ETF：沪深300 ETF
  
- 🇺🇸 **美国资产** (`EQUITY_US`)
  - 美股本土公司：苹果、特斯拉、微软
  - 投资美国的基金/ETF：标普500 ETF、纳斯达克100 ETF
  
- 🇯🇵 **日本资产** (`EQUITY_JP`)
  - 日股：丰田、索尼
  - 投资日本的基金/ETF：日经225 ETF
  
- 🇭🇰 **香港资产** (`EQUITY_HK`)
  - 香港本地公司：港交所(388)、汇丰银行(5)
  
- 🌏 **其他地区** (`EQUITY_OTHER`)
  - 欧洲：宝马、雀巢
  - 新兴市场：全球分散投资ETF

**风险特性**：
- 风险等级：高
- 流动性：中等
- 预期收益：高
- 建议占比：40-70%

---

#### 2. 🟢 现金及现金等价物 `#10B981`
**定义**：流动性最强的资产

**二级分类（按流动性）**：
- 活期存款 (`CASH_DEMAND`)
- 定期存款 (`CASH_FIXED`)
- 货币基金 (`CASH_MONEY_FUND`) - 如余额宝、零钱通
- 券商现金 (`CASH_BROKER`)

**风险特性**：
- 风险等级：低
- 流动性：高
- 预期收益：低
- 建议占比：3-10%

---

#### 3. 🟠 固定收益类 `#F59E0B`
**定义**：提供固定或可预测收益的投资

**二级分类（按类型）**：
- 债券 (`FIXED_BOND`) - 国债、公司债
- 可转债 (`FIXED_CONVERTIBLE`)
- 理财产品 (`FIXED_WEALTH`) - 银行理财、信托

**风险特性**：
- 风险等级：低
- 流动性：中等
- 预期收益：低
- 建议占比：20-40%

---

#### 4. 🔷 不动产类 `#6366F1`
**定义**：实物资产，保值增值

**二级分类（按类型）**：
- 住宅房产 (`RE_RESIDENTIAL`)
- 商业地产 (`RE_COMMERCIAL`)
- 房地产信托 (`RE_REITS`)

**风险特性**：
- 风险等级：中等
- 流动性：低
- 预期收益：中等
- 建议占比：0-50%

---

#### 5. 🌸 另类投资 `#EC4899`
**定义**：非传统投资品，分散风险

**二级分类（按类型）**：
- 贵金属 (`ALT_GOLD`) - 黄金、白银
- 大宗商品 (`ALT_COMMODITY`) - 原油、农产品
- 数字资产 (`ALT_CRYPTO`) - 比特币、以太坊
- 收藏品 (`ALT_COLLECTIBLE`) - 艺术品、古董

**风险特性**：
- 风险等级：极高
- 流动性：低
- 预期收益：高
- 建议占比：0-15%

## 🔧 技术实现

### 数据库Schema变更

#### Security 表新增字段
```prisma
model Security {
  // ... 现有字段
  
  // 新增：投资标的所属国家/地区
  targetCountry   String?  @map("target_country") @db.VarChar(10)
  
  // 说明：
  // - region: 交易市场（如 US = 美股市场）
  // - targetCountry: 投资标的所属地（如 CN = 中国公司）
}
```

### 智能判断逻辑

#### 1. 中概股识别
```javascript
const CHINESE_ADR_SYMBOLS = [
  'PDD',   // 拼多多
  'BABA',  // 阿里巴巴
  'JD',    // 京东
  'BIDU',  // 百度
  'NIO',   // 蔚来
  'XPEV',  // 小鹏
  'LI',    // 理想
  // ... 更多
];
```

#### 2. 美国基金识别
```javascript
const US_FUND_KEYWORDS = [
  '标普', 'S&P', 'SP500',
  '纳斯达克', 'NASDAQ', 'QQQ',
  '道琼斯', 'DOW',
  '美国50', 'US50', '美股'
];
```

#### 3. 判断优先级
```
1. 检查中概股列表 (CHINESE_ADR_SYMBOLS)
   ↓
2. 检查基金名称关键词 (US_FUND_KEYWORDS)
   ↓
3. 检查名称中的国家关键词 (中国/日本等)
   ↓
4. 根据交易市场推测 (region)
   ↓
5. 返回 null（待手动设置）
```

## 🎯 用户价值

### 1. 清晰的资产配置视图
```
传统视图（交易市场）:
  - 美股账户: $50,000
  - 港股账户: $30,000
  - A股账户: $20,000

V2.0 视图（经济体风险）:
  - 🇨🇳 中国资产: $70,000 (70%) ← 真实风险暴露
```

- 一目了然看到各国资产占比
- 避免"以为分散，实际集中"的风险
- 例如：港股+中概股 = 都是中国资产

### 3. 智能的投资建议
```
系统检测到:
  - 中国资产占比 85% (过高 ⚠️)
  - 美国资产占比 10%
  - 日本资产占比 5%

建议:
  - 考虑增加美国资产配置
  - 参考标准配置: 本国60% + 海外40%
```
