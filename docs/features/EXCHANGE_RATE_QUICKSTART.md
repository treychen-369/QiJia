# 汇率功能快速入门 🚀

## 30秒快速上手

### 1️⃣ 查看实时汇率
访问仪表板 → 在图表区域右侧看到"实时汇率"组件

### 2️⃣ 手动刷新汇率
点击汇率组件右上角的 🔄 刷新按钮

### 3️⃣ 查看更新时间
汇率下方显示"更新于: X分钟前"

## 常见场景

### 场景1: 查看当前汇率
**操作**: 无需操作，页面自动加载  
**频率**: 首次访问 + 每小时自动刷新

### 场景2: 需要最新汇率
**操作**: 点击 🔄 刷新按钮  
**耗时**: 2-3秒  
**结果**: 显示实时汇率

### 场景3: 汇率显示"默认"
**原因**: API请求失败  
**操作**: 点击刷新重试  
**备用**: 使用默认汇率(USD:7.2, HKD:0.92)

## 一分钟API使用

```typescript
// 1. 导入服务
import { getExchangeRates, convertCurrency } from '@/lib/exchange-rate-service'

// 2. 获取汇率
const rates = await getExchangeRates()

// 3. 转换货币
const cnyAmount = convertCurrency(100, 'USD', 'CNY', rates)
// 结果: 720 CNY (假设汇率为7.2)

// 4. 格式化显示
import { formatCurrencyAmount } from '@/lib/exchange-rate-service'
const formatted = formatCurrencyAmount(720, 'CNY')
// 结果: "¥720.00"
```

## 故障自救

### ❌ 汇率不更新
1. 点击刷新按钮
2. 刷新浏览器页面
3. 清除浏览器缓存

### ❌ 显示"加载失败"
1. 检查网络连接
2. 重新登录系统
3. 联系管理员

### ❌ 汇率显示异常
1. 查看数据来源标签
2. 如果是"默认"，点击刷新
3. 对比实际汇率验证

## 5个实用技巧

💡 **技巧1**: 每天开市前刷新汇率，确保准确性  
💡 **技巧2**: 汇率显示"实时"表示最新数据，"默认"表示备用值  
💡 **技巧3**: 系统会自动缓存1小时，避免频繁请求  
💡 **技巧4**: 更新时间显示相对时间，方便判断新鲜度  
💡 **技巧5**: 汇率组件响应式设计，手机端也可正常使用
