# 实物资产功能实现

## 📅 实现日期
2026年2月1日

## 🎯 功能概述

在"另类投资"类目下新增"实物资产"（ALT_PHYSICAL）子类目，用于记录汽车、珠宝首饰、手表等消耗性实物资产。

## 📊 数据库变更

### 新增资产分类
```sql
-- AssetCategory 表新增记录
code: 'ALT_PHYSICAL'
name: '实物资产'
description: '汽车、珠宝首饰等实物资产'
level: 2
parentId: ALTERNATIVE 的 ID
color: '#94A3B8' (slate-400 灰色系)
```

## 💡 设计说明

### 为什么放在"另类投资"下？

1. **消耗性特征**：实物资产通常会贬值（如汽车），不属于传统投资
2. **非金融资产**：不是证券、现金、固定收益等金融资产
3. **统一管理**：与收藏品、贵金属等其他实物类资产统一管理
4. **资产盘点**：便于全面记录家庭资产

### 与收藏品的区别

| 特征 | 实物资产 | 收藏品 |
|-----|---------|--------|
| 主要用途 | 使用功能 | 收藏/投资 |
| 价值趋势 | 通常贬值 | 可能升值 |
| 示例 | 代步汽车 | 古董钟表 |

## 🔮 未来优化方向

1. **折旧计算**：支持自动计算汽车等资产的折旧
2. **保险关联**：关联相关保险信息
3. **维护记录**：记录保养、维修历史
4. **估值更新**：定期更新市场估值

## ✅ 测试验证

1. 刷新页面，进入"资产详情" → "另类投资"
2. 点击"添加另类投资"按钮
3. 应看到5个资产类型选项，包括"实物资产"
4. 选择"实物资产"，填写表单
5. 验证资产正确保存和显示
