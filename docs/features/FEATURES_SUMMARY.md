## 💻 技术栈

### 前端
- **框架**: Next.js 14 + React
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **UI库**: Radix UI
- **状态**: React Hooks
- **图表**: Recharts

### 后端
- **框架**: Next.js API Routes
- **认证**: NextAuth.js
- **数据库**: PostgreSQL
- **ORM**: Prisma
- **验证**: Zod

### 部署
- **容器**: Docker + Docker Compose
- **代理**: Nginx
- **环境**: Node.js 20+
