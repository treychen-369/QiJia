# 持仓管理功能指南

## ✨ 功能概述

持仓列表模块现已完全可用，支持查看详情、编辑持仓和删除记录三大核心功能。

## 🎯 核心功能

### 1. 查看持仓详情 👀
点击持仓卡片右侧的"..."菜单 → 选择"查看详情"

**显示内容**：
- 💰 当前价格和持仓数量
- 📊 市场价值和总成本
- 📈 未实现盈亏（金额和百分比）
- 📉 今日涨跌情况
- 📋 完整的股票信息（代码、名称、地区、行业、币种）
- 🧮 盈亏计算公式说明

**特点**：
- 大屏展示，信息一目了然
- 盈亏用颜色区分（绿色盈利，红色亏损）
- 包含计算公式，帮助理解数据来源

### 2. 编辑持仓信息 ✏️
点击"..."菜单 → 选择"编辑持仓"

**可编辑字段**：
- 持仓数量（必填）
- 成本价（必填）
- 当前价格（必填）

**实时预览**：
- 市场价值 = 数量 × 当前价格
- 总成本 = 数量 × 成本价
- 未实现盈亏 = 市场价值 - 总成本
- 盈亏比例 = (盈亏 / 总成本) × 100%

**特点**：
- 实时计算预览，确认后再保存
- 表单验证，防止无效输入
- 保存后自动刷新列表

### 3. 删除持仓记录 🗑️
点击"..."菜单 → 选择"删除记录"

**安全确认**：
- 显示完整的持仓信息
- 警告提示：删除后无法恢复
- 需要二次确认

**特点**：
- 防止误删，确保数据安全
- 删除后自动刷新列表

## 📱 使用流程

### 查看详情
```
1. 在持仓列表找到目标股票
2. 鼠标悬停显示操作按钮
3. 点击"..."打开菜单
4. 选择"查看详情"
5. 查看完整信息
6. 点击关闭或按ESC键退出
```

### 编辑持仓
```
1. 打开"..."菜单
2. 选择"编辑持仓"
3. 修改数量、成本价或当前价格
4. 查看预览计算结果
5. 确认无误后点击"保存"
6. 等待保存成功提示
```

### 删除记录
```
1. 打开"..."菜单
2. 选择"删除记录"（红色选项）
3. 仔细阅读确认信息
4. 点击"确认删除"
5. 等待删除成功提示
```

## 🔧 技术实现

### API端点

#### 获取单个持仓详情
```http
GET /api/holdings/{id}
```

#### 更新持仓
```http
PUT /api/holdings/{id}
Content-Type: application/json

{
  "quantity": 1000,
  "costBasis": 50.5,
  "currentPrice": 55.2
}
```

#### 删除持仓
```http
DELETE /api/holdings/{id}
```

## 🛡️ 安全机制

### 权限验证
- API端点需要NextAuth会话
- 验证持仓所属用户
- 防止越权访问

### 数据验证
- 持仓数量必须 > 0
- 成本价必须 > 0
- 当前价格必须 > 0
- 后端二次验证

### 错误处理
- API调用失败显示Toast提示
- 表单验证错误显示在字段下方
- 网络错误自动重试（用户手动）

## 🐛 常见问题

### Q1: 为什么操作按钮看不见？
**A**: 操作按钮需要鼠标悬停到持仓卡片上才会显示，这是为了保持界面简洁。

### Q2: 编辑后数据没有更新？
**A**: 
1. 检查网络连接
2. 查看浏览器控制台是否有错误
3. 刷新页面重试
4. 确认已登录系统

### Q3: 删除持仓后能恢复吗？
**A**: 不能。删除操作不可逆，请谨慎操作。建议先导出数据备份。

### Q4: 编辑时预览计算不准确？
**A**: 预览计算是实时的，确保输入的都是数字。如果出现问题，刷新页面重试。

### Q5: 保存时提示"未授权访问"？
**A**: 会话可能已过期，请重新登录系统。
