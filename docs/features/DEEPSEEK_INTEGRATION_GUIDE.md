# DeepSeek AI 集成指南

## 📋 功能概述

本系统已集成 DeepSeek AI，用于生成个性化的家庭资产配置建议。

### 核心功能

1. **动态提示词生成**：根据用户的家庭概况和当前资产配置自动生成优化的提示词
2. **提示词预览和编辑**：用户可以查看、修改提示词以获得更精准的建议
3. **AI 资产配置建议**：基于 DeepSeek-V3.2 模型生成专业的配置建议
4. **降级策略**：当 AI 服务不可用时自动切换到规则引擎

---

## ⚙️ 关键参数配置

### 1. Model 选择

| 模型 | 说明 | 适合场景 |
|------|------|---------|
| `deepseek-chat` ✅ | DeepSeek-V3.2 **非思考模式** | 数据分析、财务建议、标准任务 |
| `deepseek-reasoner` | DeepSeek-V3.2 **思考模式** | 复杂数学证明、多步推理 |

**当前配置**：`deepseek-chat`

**选择理由**：
- 财务建议场景输入结构清晰，输出格式固定（JSON）
- 不需要复杂的多轮内部推理
- 非思考模式响应更快、成本更低

### 2. Temperature 选择

根据 DeepSeek 官方建议：

| 场景 | 推荐 Temperature |
|------|-----------------|
| 代码生成/数学解题 | 0.0 |
| **数据抽取/分析** | **1.0** |
| 通用对话 | 1.3 |
| 翻译 | 1.3 |
| 创意类写作 | 1.5 |

**当前配置**：`0.8`

**选择理由**：
- 财务建议属于"数据分析+决策建议"的混合场景
- 官方推荐数据分析使用 1.0
- 但财务建议需要一定稳定性（相同输入应产生相似输出）
- 取 0.8 作为平衡点

## 🎯 使用流程

### 1. 填写家庭概况

在"资产配置管理"页面的"家庭概况"标签页中填写：

- 家庭成员信息
- 收入与支出
- 风险偏好
- 投资期限
- 负债情况
- 保险保障

### 2. 请求 AI 建议

点击"请求AI建议"按钮，打开提示词编辑器：

- **预览模式**：查看自动生成的提示词
- **编辑模式**：可自定义修改提示词
- **补充说明**：添加额外信息供 AI 参考

### 3. 获取建议

点击"获取 AI 建议"后，系统将：

1. 调用 DeepSeek API
2. 解析返回的 JSON 建议
3. 显示调仓建议和风险提示
4. 保存建议记录

### 4. 采纳建议

查看建议后可选择"采纳建议"，系统将自动更新配置目标。

## 🔄 降级策略

当 DeepSeek API 不可用时（网络错误、API Key 失效等），系统自动切换到内置规则引擎：

### 规则引擎逻辑

1. **年龄因素**
   - < 30岁：权益类 60%，现金 5%
   - 30-50岁：权益类 50%，现金 10%
   - > 50岁：权益类 30%，固收 40%，现金 15%

2. **风险偏好调整**
   - 保守型：权益类 -15%，固收 +10%，现金 +5%
   - 激进型：权益类 +10%，固收 -10%

3. **应急资金检查**
   - 检查现金类是否覆盖 6 个月支出
   - 不足时优先建议增配现金

## 🐛 常见问题

### Q: API 返回错误 401

**原因**：API Key 无效或已过期

**解决**：
1. 检查 `.env.local` 中的 `DEEPSEEK_API_KEY`
2. 访问 [DeepSeek 开放平台](https://platform.deepseek.com/) 重新生成

### Q: 建议格式解析失败

**原因**：AI 返回的 JSON 格式不完整

**解决**：系统会自动降级到规则引擎，同时在控制台记录原始响应

### Q: 请求超时

**原因**：网络问题或 API 响应慢

**解决**：
1. 检查网络连接
2. 重试请求
3. 如持续超时，系统会自动使用规则引擎
