# 定期存款功能说明

## 📊 收益计算逻辑

### 方案设计

定期存款采用**实时累计收益**的方式，让您随时了解当前的收益状况。

### 收益计算公式

```
当前累计利息 = 本金 × 年利率 × 已持有天数 / 365
当前价值 = 本金 + 当前累计利息
```

### 示例

假设您有一笔定期存款：
- **本金**：50万元
- **年利率**：2.13%
- **存入日期**：2026年1月1日
- **到期日期**：2028年3月3日（共793天）
- **当前日期**：2026年1月25日（已持有24天）

**计算过程**：
```
预计总利息 = 500,000 × 0.0213 × 793 / 365 = 23,167.40元
当前累计利息 = 500,000 × 0.0213 × 24 / 365 = 701.30元
当前价值 = 500,000 + 701.30 = 500,701.30元
```

## ✨ 功能特点

### 1. 添加定期存款

**必填字段**：
- 资产名称：如"南洋银行-定期"
- 金额：本金
- 银行名称：开户银行
- 年利率：百分比形式（如 2.13）
- 存入日期
- 到期日期

**可选字段**：
- 账户后四位
- 备注说明

**实时预览**：
- 填写完毕后，表单会自动显示：
  - 本金
  - 预计利息
  - 到期本息
  - 存款天数

### 2. 资产列表显示

**动态计算**：
- 系统每次打开都会**自动计算当天的累计收益**
- 无需手动更新

**显示内容**：
- 当前价值 = 本金 + 累计利息
- 累计收益（绿色显示）
- 收益率百分比

### 3. 到期处理

**未到期时**：
- 显示实时累计收益
- 根据已持有天数按比例计算

**已到期后**：
- 自动显示完整的预计收益
- 可选择性填写**实际到期本息**（如果和预计不同）

## 🎯 典型使用场景

### 场景3：存款到期

- 系统自动显示已到期

- 可选择性填写实际到期金额（如果不同）

## 💡 注意事项

### 计算方式
- 使用**单利计算**（大多数定期存款）
- 按**实际天数/365**计算
- 不考虑提前支取、自动续存等复杂情况

### 收益显示
- **未到期**：显示当前累计收益（每天增加）
- **已到期**：显示完整收益
- 收益实时更新，无需手动刷新

### 数据准确性
- 预计收益仅供参考
- 实际收益以银行结算为准
- 建议到期后填写实际金额

## 🔧 技术实现

### 前端
- 实时计算组件：`fixed-deposit-calculator.ts`
- 自动预览：表单中实时显示预计收益
- 列表展示：动态计算并显示当前价值

### 后端
- 数据存储：`metadata` JSON字段
- 利率：`metadata.interestRate`
- 实际到期本息：`metadata.actualMaturityAmount`（可选）

### 公式
```typescript
const interest = principal * rate * days / 365;
const currentValue = principal + interest;
```
